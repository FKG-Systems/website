# FKG Systems Website

A cinematic static landing page for FKG Systems / FKGAL Aviation.

## Local preview

```bash
python3 -m http.server 5173
```

Open: http://127.0.0.1:5173

## Contents

- `index.html` — site structure and copy
- `styles.css` — responsive visual design
- `script.js` — header state and reveal animations

The current imagery is pulled from the public FKG-Systems/Wwbsite-logolar repository so the website stays lightweight.
